Social Media AI Copy

Connection

Shared by all AI functions. Find these values in the Azure Portal under your Azure OpenAI resource.

Your Azure OpenAI resource key → Keys and Endpoint.
Base URL of your resource. Format: https://<your-resource>.openai.azure.com

Text Generation

Used for both "Rephrase with AI" and "Create Website Copy". Deploy a GPT-4o model in Azure AI Foundry.

The name of your text model deployment (e.g. gpt-4o). Not the model name — the deployment name.
Use 2024-02-01 or later for GPT-4o. See API reference.
Instructions for the Rephrase with AI context menu action. The selected text is appended automatically.
Instructions for the Create Social Media Copy for this Website action. Page title, URL, and content are appended automatically. Use {headline} if you want to reference the selected quote.

Image Generation

Uses the same API Key and Endpoint as above. Deploy a gpt-image-1 model in Azure AI Foundry. Leave Deployment Name blank to hide the Create Image button.

The name of your gpt-image-1 deployment.
Instructions for the promotional image. Use {headline} where you want the article headline or selected text to appear.
